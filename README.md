# Supermarket Product Deduplication & Knowledge Graph (Neo4j)

## Overview
Built an end-to-end data pipeline to clean, standardize, and deduplicate supermarket product data, modeling it into a Neo4j knowledge graph that enables unified product search and cross-retailer price comparison across multiple New Zealand supermarkets.

## Workflow & Tools
- Cleaned and consolidated raw data using **Excel** and **Power Query**
  - Removed duplicates (~20,000 → ~700 records)
  - Standardized text fields and identifiers
- Built a **Neo4j knowledge graph** with nodes: Product, Category, Brand, Listing, Store
- Designed relationships to link equivalent products across stores
- Enabled flexible querying for price, brand, category, and store-specific comparisons

## Key Outcomes
- Reduced raw data by 96% through deduplication
- Created a scalable graph model for cross-store product comparison

## Skills
Data cleaning, deduplication, Excel, Power Query, Neo4j, knowledge graph modeling, entity resolution
